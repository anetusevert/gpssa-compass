# GPSSA Intelligence — Project Playbook

**RFP:** GPSSA-016-2026 · Product & Service Development Roadmap and Quality Assurance Framework

## What this is for

Run the dual workstream engagement: **Workstream A** (12-month product & service roadmap) and **Workstream B** (end-to-end QA & fulfilment). Compass is the shared working file for workshops and decisions.

## What this is not

- Not a live GPSSA case / CRM / Ma’ashi system of record
- Not a finished production QA platform
- Not a museum of 30 screens to explore randomly

## Three jobs only

1. **Diagnose the estate** — catalogue, channels, mandate/RFP anchors, known gaps
2. **Decide the roadmap** — opportunities → ranked backlog → 12-month plan with owners
3. **Design / pilot QA & fulfilment** — scorecards, cases, SLA/breach, then handover

## Where to start this week

Open **Home → Engagement Mode → Discover (W1–4)**.

Work only the three to five screens listed for that phase. Ignore the rest of the sidebar until you switch phase.

| Phase | Weeks | RFP | Open these |
|---|---|---|---|
| Discover | 1–4 | A1 + B1 | Service Catalog, Channels, RFP Alignment, QA Framework, Fulfilment Cases |
| Evidence | 2–5 | A2 | Breach, Fulfilment Analytics, VoC, Benefits |
| Shape | 6–14 | A3–A4 + B2 | Opportunity Backlog, QA Scorecards, Reviews, Products Portfolio |
| Lock | 13–17 | A5 + B3 | 12-Month Roadmap, Pilot reviews, Case Board |

## Demo vs client data

Quality, Fulfilment, Performance, and Planning ship with **gold seed** so walkthroughs are never empty. Treat seeded numbers as rehearsal until client evidence is imported. A banner on those modules says so explicitly.

## Soft workflow

- Opportunities: set **owner**, **status**, and **Log decision** when the room agrees
- CAPA: keep **owner** + **status** current
- Import CSV opportunities via Admin → Data when replacing seed

## Walkthrough success criteria (Phase 4)

Use on the next colleague / sponsor walkthrough:

1. New user reaches first useful screen in **under 2 minutes** without asking “where do I start?”
2. After Engagement Mode once, they can state the **three jobs** unprompted
3. They never confuse **seed vs client** data on ops pages (banner visible)
4. Workshop can run **Discover → 3 screens → one logged opportunity** without opening Atlas or Briefing

## One sentence

Use it to run workshops and decisions — catalogue → evidence → opportunities → roadmap → QA/fulfilment controls — with every call tied to RFP GPSSA-016-2026.
